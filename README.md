## Clase 08/05
### ¿Que es la infomatica?**
Todos los desarrollos a partir del siglo 20, herramientas para desarrollar, procesar y transmitir datos
### ¿Qué es Hardware?
La parte tagible de las computadoras: Hard (duro), Ware (Componente):
Elementos como un monitor, teclado, procesador, ram, alamacenamiento y placa madre.
 - Existe hardware complementario: como las tarjetas de expansión como red, sonido, video, etc.
 - Dispositivos tanto de entrada y salida: "**ejemplos**"
 - **Componentes:** CPU, ram, placa base, disco duro
#### Puertos
* Puntos fisicos o virtuales donde se conectan los perifericos
* Puertos fisicos -> PCI, VGA, USB, Ethernet
### ¿Qué es el software?
La parte no tangible de las computadoras: soft (suave), ware (componente):
Cosas como los programas, las licencias, etc.
### RS232 (serial port)
* **Identificación:** Conector DB9 (9 pines, forma de 'D')
* **Uso Biomédico:** Para el personal de servicio, es un Estandar en equipos antiguos (monitores de signos vitales, ventiladores, máquinas, etc)
* Es un puerto sensible al voltaje, y equiere cuidados especiales
###  Protocolo RS232 (Legacy Support)
* **Arquitectura:** Utiliza niveles de voltaje entre $+3V$  y $+15V$ para el "0" lógico y $-3V$ a $-15V$ para el "1".
* **Punto Crítico:** A diferencia del USB, el RS232 no es Plug-and-Play. Si se conecta con el equipo encendido y hay una diferencia de potencial (tierra) entre la PC y el monitor

* Un técnico debe realizar una "Lista de Verificación de Hardware" antes de asumir que el software es el culpable:
* **Verificación de cableado:** ¿El cable está blindado? (Crucial en entornos biomédicos para evitar interferencia electromagnética).
* **Estado de los conectores:** Verificar pines doblados o corrosión
